## BinGen

```commandline
bingen opts

Usage:
  bingen [command]

Available Commands:
  gen         generate needed executables for your app
  help        Help about any command

Flags:
      --add stringArray   Add new tools
      --build             Build all tools
  -h, --help              Help for the CLI
  -v, --verbose           Verbose level output
      --version           Print the CLI version

Use "bingen [command] --help" for more information about a command.

```
